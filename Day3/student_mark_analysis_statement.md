**Python Program – Student Marks Analysis**

### **Problem Statement**

Write a **Python program** to analyze student marks using the requirements given below.

---

###  **Functional Requirements**

#### **Input**

* Read marks of **N students** (integer values).
* Marks should be in the range **0 to 100**.

---

#### **Processing**

* Use a **loop** to process all student marks.

* Use **if – elif – else** conditions to classify each student as:

  * **Pass** → marks ≥ 40
  * **Fail** → marks < 40

* Calculate the following:

  * Total marks
  * Average marks
  * Number of passed students
  * Number of failed students

---

#### **Operators (Mandatory Usage)**

Use the following operators in your program:

* Arithmetic operators: `+`, `/`
* Comparison operators: `>=`, `<`
* Assignment operators: `+=`

---

###  **Output**

Display the following results:

* Total marks
* Average marks
* Number of passed students
* Number of failed students

---


###  **Sample Output (Example)**

```
Total Marks: 350
Average Marks: 70.0
Passed Students: 4
Failed Students: 1
```

---

###  **Code Quality Requirements (Clean Code)**

Your program **must follow clean code principles**:

* Use **meaningful variable names**
* Follow **proper indentation**
* Write **separate functions** where required
* Add **comments explaining WHY the logic is used**, not WHAT the code does
* Avoid hard-coded values (use constants)

---

* **Time Complexity**
